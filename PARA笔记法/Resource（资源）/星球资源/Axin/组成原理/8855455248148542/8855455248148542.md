怎么理解：1、DMA以数据块为基本传送单位 
                    2、DMA接口与主存之间的传送单位为字，DMA与设备之间的传送单位可能为字节或者位
这些都是系统根据实际需要设计出来的规则。

1、DMA以数据块为基本传送单位【主要讲的是传输过程】:
- DMA(Direct Memory Access)是一种可以直接在主存和I/O设备之间传输数据的机制,无需CPU参与。
- 为了提高传输效率,DMA通常是以数据块(如一页、一个扇区等)为基本传送单位,而不是单个字或字节。这样可以减少DMA控制器频繁启动和停止的开销,提高数据传输速度。

2、DMA接口与主存之间的访存单位为字,DMA与设备之间的访存单位可能为字节或者位【主要讲的是传输前后的读写过程】:
- 为了与主存进行高效的数据交换,DMA控制器通常以字(word)为单位访问主存。这是因为主存的基本访问单位是字。
- 但是,DMA控制器与I/O设备之间的数据传输单位可能是字节(byte)或者甚至是位(bit)，这是因为不同的I/O设备可能有不同的数据位宽要求,DMA控制器需要适配这些差异。
