            <div class="content ql-editor"><h3>详细说一下这四个线的区别，防止大家混淆</h3><p><br></p><h3><strong style="background-color: rgb(255, 255, 255); color: rgb(18, 18, 18);">1、国家线</strong></h3><p><br></p><p>考研国家线是教育部依据硕士生培养目标，结合年度招生计划、生源情况及总体初试成绩情况，确定报考统考、MBA及法律硕士专业学位考生进入复试的<strong>基本要求标准</strong>，其中包括<strong>应试科目总分要求</strong>和<strong>单科分数要求</strong>。应届本科毕业生和非应届毕业生实行<strong>统一的进入复试基本分数要求</strong>。亦是指考生口中所说的"考研国家分数线"、"全国考研分数线"、"考研国家线"、"国家复试分数线"、"国家分数线"、"考研国家复试基本分数线"。分为专业课、公共课两种，依据不同地区，分数线标准有所差异，是考研调剂的最重要依据。</p><p><br></p><p>考研国家线划分为<strong>A、B</strong>两类，<strong>A类</strong>分数线最高，其次是<strong>B类地区</strong>。这两类地区的分数线通常会相差<strong>2~10</strong>分。</p><p><br></p><p>分区概念:国家按照一区、二区确定考生参加复试基本分数要求，</p><p><br></p><p>一区包括北京、天津、上海、江苏、浙江、福建、山东、河南、湖北、湖南、广东、河北、山西、辽宁、吉林、黑龙江、安徽、江西、重庆、四川、陕西等21省(市);</p><p><br></p><p>二区包括内蒙古、广西、海南、贵州、云南、西藏、甘肃、青海、宁夏、新疆等10省(区)。</p><p><br></p><h3><strong style="background-color: rgb(255, 255, 255); color: rgb(18, 18, 18);">2、34所自划线</strong></h3><p><br></p><p>“34所”是指考研中34所自主划线的院校，自主划定分数线是2002年才开始的。国家出台这个政策的原意是让学校自己选择人才培养。具体的方式就是，初试的分数线由各个有自主划线权的学校自己划定。而不用参考国家统一划定的分数线。另外需要强调的一点是，34所院校都是985院校，只有这34所具备自主划线的权利。一般来说分数线要高于国家线，甚至有些专业高出国家线很多很多。只有极少数情况，可能自主划线会低于国家线。</p><p><br></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(51, 51, 51);">34所具体有：北京大学、清华大学、中国人民大学、北京师范大学、北京航空航天大学、北京理工大学、中国农业大学、上海交通大学、复旦大学、同济大学、南开大学、天津大学、中国科学技术大学、哈尔滨工业大学、吉林大学、东北大学、大连理工大学、山东大学、南京大学、东南大学、浙江大学、厦门大学、华中科技大学、武汉大学、湖南大学、中南大学、西安交通大学、西北工业大学、兰州大学、中山大学、华南理工大学、电子科技大学、重庆大学、四川大学。</span></p><p><br></p><h3><strong style="background-color: rgb(255, 255, 255); color: rgb(18, 18, 18);">3、复试分数线</strong></h3><p><br></p><p>各招生单位在<strong>国家线</strong>的基础上，<strong>根据本校</strong>有关专业生源余缺确定的复试资格线，<strong>等于或者高于国家线</strong>。</p><p><br></p><h3><strong style="background-color: rgb(255, 255, 255); color: rgb(18, 18, 18);">4、录取线</strong></h3><p><br></p><p>可以从<strong>拟录取名单</strong>当中找到的该专业<strong>最后一名</strong>的<strong>分数</strong>。因为考研复试都是1:2或者1:5差额的形式，所以我们不仅要关注复试线，还要去拟录取名单上找到最后一名的分数，以这个分数作为自己的底线目标。</p><p><br></p><p>1:2或者1:5差额是指：你报考的学校有10个录取名额，那么就让12或者15个人进复试，然后再淘汰2-5个。这就是复试差额比，第12个人的初试分数就是该专业的复试分数线。</p><p><br></p></div>
