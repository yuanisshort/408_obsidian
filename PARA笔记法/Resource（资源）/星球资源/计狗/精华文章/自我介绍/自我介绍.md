            <div class="content ql-editor"><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">1⃣️自我介绍 陕西国防学院，软件技术专业，23届专科毕业生</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">大一学的状态和很多大学生一样，感受着大学生活的美好，同时编程上处于懵懵懂懂的状态，在大一下半学期知道学校有专门的技术培养基地，参与考核并通过，在刚进入时觉得这里的环境超级好，有着舒服的靠椅和学校最好的电脑，拥有1年的使用期，于是大二一整年处于编程能力的上升期，选择了移动应用工坊，在经历了半学期后因要考学校当时的一个Java中级证书未能通过海选报名，果断决定选择Java作为未来的发展道路。在大一下半学期参与自考，当时本着快速考过的原则，知识好获取的原则选择了工商管理，后来没想到是天坑专业，考研都要求太多。</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">2022年初通过了证书海选和报名考试，拥有了第一个编程证书，由于我在学校实力不够看，想突破现状，于是寻找各种校外编程比赛参加，和山东的学校几个同学参与中国计算机大赛顺利通过初赛。</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">2022年夏天打算一边找工作一边准备升本考试，起码得有经验，了解当前市场需求，工作竞争力。并于放暑假的第三天7.3号顺利在家人工作附近找到一个小公司的Java开发实习，所幸有大哥愿意带我还好一点，公司实习期并没有安排过多任务，大多都是一些数据库的建表改数据、写文档之类的小活。每天大哥一边带我我一边学习更深入的东西doctor,微服务、各种设计模式，顺带着了解公司当前项目的业务情况，本以为后面日子会越来越好，可是好景不长，在7.20日公司开始安排了一个后端管理系统的项目，让我单独搞，可是我研究了它的前端和我之前练习项目时的前端写法相差较大，请教大哥，大哥说用ajax写，我搜索了半天不知道juquery怎么调用后端接口，后来成功调用了，可是页面原有的一些格式发生了变动，我觉得我完蛋了！！后面正好我的中国高校计算机大赛进入了复赛，比赛组织方邀请我团队去集训5天，我索性就向老板请了假，将项目打包交给了别的同事。心想着完就完了，就这样我去了浙大参加集训，开阔了眼界，萌生了考研的念头，后续比赛拿了国三，同时工作也黄了，回来后工位已经是一个测试的女生，第二天人事就通知我该走了。</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">此刻已经是2022.8.6，在后续一个月中我一边找工作一边搞着比赛，比赛的决赛我没有去，一边是找到了新工作大公司的运维岗位，一边因为疫情还有路程时间久线上完成了比赛，得了国三。后面在大公司干税务运维，由于在国企驻地，一切都还好，后面有点不甘心想干开发，还有就是工资低就干了一个月跑回学校备考专升本了，后续考上民办了不甘心放弃了，家里农村出身经济不好于是找工作。</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">2023.7月找到了个比亚迪的Mes运维岗位，发现与所学知识严重不匹配，更像是实施工作，工作不好找我默默忍受着。过去了一月和领导工作意见不统一，工资少，又跑路了，现在在重庆某软件公司干运维，打算边工作边考研。希望上岸。自考本科因为疫情仅仅过了6门课程，不知道要不要继续下去？还是转到自考计算机上来。</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"><br></span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">正在备考22408。 </span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"> </span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);"> 2⃣️备考情况（22408） ⭐️数学：已完成——高数基础， ⭐️408：已完成——王道c视频课20节+课后选择题。</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">3. 院校选择:本来因为比赛想选择浙大，但不敢，本身英语基础薄弱，平时最多考60分那种的</span></p><p><span style="background-color: rgb(255, 255, 255); color: rgb(128, 128, 128);">地区一：杭州，选择杭州师范大学</span></p><p><span style="background-color: rgb(255, 255, 255);
