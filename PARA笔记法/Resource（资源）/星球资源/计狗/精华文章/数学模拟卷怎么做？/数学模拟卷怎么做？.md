            <div class="content ql-editor"><p><span style="color: rgb(217, 217, 217); font-size: 15px;">转载自——为你代研数学团队</span></p><h1 class="ql-align-justify"><span style="background-color: rgb(255, 255, 255); font-size: 15px; color: rgb(89, 89, 89);">今天我们主要聊聊模拟题到底应该怎么做？</span></h1><p><br></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">其实模拟题就两个作用。</span></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">第一检验，检验你现在自己的考试状态，检验自己的分数水平，第二就是查漏补缺，帮你找问题。</span></p><p><br></p><p><strong>（一）考试中的技术问题</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">技术问题包括做题顺序和精力管理，还有就是时间分配。</strong></p><p><strong style="background-color: rgb(239, 112, 96); color: rgb(255, 255, 255);">1、做题顺序：</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">一般是从头到尾去做</strong><span style="font-size: 15px; color: rgb(89, 89, 89);">，但有的同学要考虑自己的情况，比如说</span><strong style="font-size: 15px; color: rgb(89, 89, 89);">有同学发现小题总是挺难的，总会有一些意想不到的方法或者技巧，容易卡在这儿</strong><span style="font-size: 15px; color: rgb(89, 89, 89);">，这时对你来说，如果你的心态容易受到这种影响。</span></p><p><strong style="font-size: 15px; color: rgb(79, 129, 189);">我们建议你可以先反过来做。</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">你先看大题，</span><strong style="font-size: 15px; color: rgb(79, 129, 189);">哪些题是一看你就会的</strong><span style="font-size: 15px; color: rgb(89, 89, 89);">，就那种比较死板的那种题目，</span><strong style="font-size: 15px; color: rgb(79, 129, 189);">你先去把它写了</strong><span style="font-size: 15px; color: rgb(89, 89, 89);">，当做热身，很多同学上考场的时候脑是懵的，他没有调动自己的精神状态的这习惯，把这种题做出来，慢慢的你的心态也平复了，状态也进入考场的状态了，这个时候再去攻克那种小题，可能会比你一开始就拿到这种难题来说，可能要安全。</span></p><p><strong style="background-color: rgb(239, 112, 96); color: rgb(255, 255, 255);">2、精力管理：</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">有同学可能做着做着就后面不行了，其实跟做题顺序是一样的。</span></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">有同学说我选填做完之后我的命就丢了一半了，我感觉我已经被掏空了，还有那么多大题，我可能做不下去，后面错误率就会非常的高，</span><strong style="font-size: 15px; color: rgb(79, 129, 189);">同样对你来说也是一样，必须要适应三个小时全神贯注，这是我们基本能力。</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">第二就是你做题顺序上，也可以去先挑简单的，你觉得不太费精力的事情，把它做了，把最费精力的东西放在最后。</strong></p><p><strong style="background-color: rgb(239, 112, 96); color: rgb(255, 255, 255);">3、时间分配：</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">选择填空要做多少时间？</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">那个大题要写多少时间？</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">我怎么去安排我的时间？</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">我怎么去看出现在我的时间是否是合理的等等，这个东西都是你在做模拟题的时候需要去把它找到的。</span></p><p><br></p><p><strong>（二）查漏补缺</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">就是你不会做，需要去想的，包括以下几类</span></p><p><strong style="background-color: rgb(239, 112, 96); color: rgb(255, 255, 255);">1、遗忘</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">那些你可能以前会做的题，现在不会做了，或者说现在可能差一点就忘了的题目，</span><strong style="font-size: 15px; color: rgb(79, 129, 189);">一定要再去回忆自己的思维导图。</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">这个模拟题就是对这种题目非常非常好的回忆，它不断的去重复，去找到那些你的盲点。</strong></p><p><strong style="background-color: rgb(239, 112, 96); color: rgb(255, 255, 255);">2、你缺少的东西</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">缺少的东西</span><strong style="font-size: 15px; color: rgb(79, 129, 189);">包括你的思维，题型和你在想问题的时候你的技巧性</strong><span style="font-size: 15px; color: rgb(89, 89, 89);">，你在做模拟题的时候，你也可以做一个补充，但这绝对不是最重要的事，这个部分你作为补充就可以了，</span><strong style="font-size: 15px; color: rgb(89, 89, 89);">一套模拟题能有那么一到两个题，给你这样的启发，就是非常非常好的，千万不要每道题都是这样，那你做起来就太累了。</strong></p><p><strong style="background-color: rgb(239, 112, 96); color: rgb(255, 255, 255);">3、一定要去练习自己的计算能力</strong></p><p><strong style="font-size: 15px; color: rgb(79, 129, 189);">一定要去练习自己计算能力</strong><span style="font-size: 15px; color: rgb(89, 89, 89);">，千万不要说每个卷子我要给自己多少时间去检查，不一定每个同学都有这样这样的时间，你按照自己的进度来，如果你没有时间去检查，那你一定要提高自己的第一次计算能力，非常重要，如果你的计算，计算力只有80%，就意味着你120分的水平只能考96分。</span></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">你觉得分数你能接受吗？肯定不行，所以说这个一定要给提上去。</span></p><p><strong style="font-size: 15px; color: rgb(79, 129, 189);">所以最后再次建议大家对自己的复习要有战略性定位</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">明确哪些考点我应该放在后面甚至舍弃，而把确保通过反复练习能得分的地方，通过练习，保证自己能够掌握住。</strong></p><p><strong style="background-color: rgb(240, 111, 89); font-size: 15px; color: rgb(255, 255, 255);">再在有限时间内做最有“性价比”的事情，把主要精力用在最容易上分的地方。</strong></p><p><strong style="font-size: 15px; color: rgb(89, 89, 89);">掌握历年真题统计的考频高，分值高，短时间能掌握的考点考法以及对应的经典例题。</strong></p><p><span style="font-size: 15px; color: rgb(89, 89, 89);">模拟题题目训练同理，不要贪多，贪难，明确我们的核心目标是什么。</span></p></div>
