
# 深圳大学2023年硕士研究生录取情况发布

人工智能（建规学院）：均分315.6

人工智能（金融科技学院）；均分312.9

计算机技术（光明实验室）：均分329.6

人工智能（光明实验室）：均分323.6

生物医学工程（医学部-生医学中心）：均分312.3

新一代电子信息技术（电信学院）：均分319

0831生物医学工程（应用技术学院）：均分303.3
