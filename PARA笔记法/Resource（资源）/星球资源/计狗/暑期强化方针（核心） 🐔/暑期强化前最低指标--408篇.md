
# 暑期强化前最低指标--408篇

Hello，大家好，我是@小林兄，为了让大家能够最大程度地利用好暑期强化这个黄金时间，这里我给大家定了一个进入强化阶段的最低指标，即暑期开始前至少要完成这些任务！由于408复习的个体差异性和时间线的层次不齐，我暂且将开始复习的时间线定为4月份开始全面复习。
1. 完成《数据结构》+《计算机组成原理》的第一轮复习。看完王道课这两门课程的所有视频，并且认真全面的完成王道书选择题（不需要做大题），同时做好错题的标注，为第二轮的强化做好准备。1. 《数据结构》**：对于算法**，要能够在和书本结合的情况下可以模拟算法的进行；**对于数据结构**，要做到掌握书中出现的数据结构能够应用的场景，同时对比相似的数据结构，能够分析他们的相同点和不同点以及适用的情况。排序算法是重中之重，在第一轮要给到足够的重视，外部排序算法第一轮只需要了解即可。1. 《计算机组成原理》：这门课难度稍大，第一轮复习只要做到对计算机的运行机制有初步的认识即可，这门课需要在第二轮结合**习题**和**《操作系统》**之后才能做到真正的掌握。在第一轮复习结束之后：大致掌握第二章的数据表示（原、反、补、浮点），比较难以攻克的**程序的机器级**表示和**CPU指令流水**，这些知识点在第一轮如果觉得它太难**，**可以给自己放轻松，这些知识之需要大致略读一下即可，真正集大成是在第二轮。如果你还在第一轮《计算机组成原理》复习当中挣扎，请不要给自己太大的压力，不要因为自我要求和自我怀疑在这耽搁太多时间，因为这门课突出的就是综合，在你知识体系尚未完全建立之前，并不是它难，而是我们还见的不够多，当我们的复习进行到后期，相信你一定可以信手拈来～
