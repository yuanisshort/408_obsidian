
# 暑期强化前最低指标---英语篇

       Hello，大家好，我是**星火**，为了让大家能够最大程度地利用好暑期强化这个黄金时间，这里我给大家定了一个进入强化阶段的最低指标，即暑期强化前至少要完成这些任务！
1. 学习阅读技巧，至少刷完04-09年英语阅读**（做完后一定要精析）**，并在这个过程中内化技巧！1. 结束小作文课程学习！（听课即可）1. 单词是完成阅读理解最重要的部分，所以希望大家在暑期开始前能否实现**尽可能多地背单词**，目的是不浪费后面的每一篇阅读理解，大家都知道我们英语只有真题能拿来练手，所以，**我给大家定的目标是app上新学+复习400个左右**。1. 在校学生认真完成期末作业，不可以挂科，希望成绩单好看的同学，更要认真完成期末作业。
