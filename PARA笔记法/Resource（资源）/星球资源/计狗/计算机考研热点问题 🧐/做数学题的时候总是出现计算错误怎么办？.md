
# 做数学题的时候总是出现计算错误怎么办？

各位好！我是**@星火**，今天想跟大家分享一下我在考研过程中**如何克服做题总是出现计算错误的问题**。大家现在处于强化末期，相信各位也刷了不少题，指定会出现明明题目分析透彻了但最后计算出错的问题。我当时在意这个问题大概是在刷真题和模拟题的时候，害怕自己**在考场上出现这样的低级错误，不仅浪费了大量时间，最终还拿不到分数**，这对努力了一年的我们来说，实在太残忍。接下来，我会先分析一下我们出现计算错误的原因，以及解决办法（本人亲测有效，在考场上存在降智行为的前提下，数二拿到了138分）。

<br>

## 原因

<br>
1. **计算量大**（**除16年真题外，其他年份的真题计算量还在正常范围内，但我建议大家接下来刷真题的时候尽量压缩时间，在2个半小时内完成，多刷模拟题，模拟题的计算量大，我们要严格要求自己，尽量避免考场上的降智行为**）

2. **对题目不熟悉，所以解起来吃力****（****拿积分举例子，如果我们没有提前学过如何解这类的积分/不熟悉此类积分的解法，那我们只能在考场上自己摸索解法，我们本身就没有自信解出来这道题，加之计算复杂，那很有可能计算出错，如此就更看不出这道题的解法****）
3. **1. **书写潦草****（这个太低级了，希望大家能自己规避，在考场上合理利用草稿纸，****还有一点提醒大家，考场上的草稿纸在你的桌子上只能有一张，如果第一张用完了想用新的草稿纸，那么要打报告，监考老师会把旧的草稿纸收走！！！所以我们一定要合理利用草稿纸，尽量不要换，草稿纸上标好题号，另外，咱们的卷子也是可以打草稿的，一些简单计算可以在那上面完成****）
4. **1. **计算跳步****（这个也希望大家自觉规避，****别太盲目自信！！！咱们都不是天才，一步一步解题不丢人****）
5. **1. **注意力不集中****（****这个我很惭愧，到目前没找到比较好的解决办法，我只能靠提高做题速度来缓解，唯一一次做卷子没走神就是在考研考场上哈哈哈哈，我当时在刷模拟卷后期都放弃克服这个问题了，我想着把走神做卷子还不出错练到极致好啦~****）**
<br>

<br>

## **解决办法**

<br>
3. **计算不跳步****（****规范书写，工整书写，哪怕在草稿纸上演算****）**1. **回查法！！！****（****这是****我最想介绍的方法****，写完这一步，回查到上一步，以免出错，这个浪费不了多少时间，一道题能有多少步骤，往大了算有20步，我们回查一下花1.5s，那么一道题多花30s而已，跟拿不到分数相比，这点时间时我们该花的！！！****）**1. **大家可以把计算出错的地方记录在本子上****，考试之前看看自己之前在哪里跌倒过，这些地方被我们大脑重点标记，在考场上就不会出现类似行为了。**
