
# 百日冲刺广播预告贴

Hello，大家好，我是**星火**。之前答应大家在9月份上线的百日冲刺规划广播即将在**明天（9月10号，周日）**和大家见面了。百日冲刺广播是**按周**进行广播，**前一周周日**会有**负责人**把大家接下来一周各科**每天需要做的任务**进行广播，同时会**广播各科规划的****建议用时****及一些****注意事项**。**希望同学们每周在冲刺广播贴下写下****本周的总结和反思****，学长学姐会根据大家的反思给出相应建议，帮助大家改进。****接下来各科会陆续进入冲刺阶段，百日冲刺广播贴主要起到对大家督促的作用，帮大家站好最后一班岗。**

由于各科情况不同，接下来我会为大家说明各科百日冲刺广播贴具体跟大家见面的时间及各位同学能够使用广播贴的前提条件。
1. **数学一：**使用数一百日冲刺手册的**前提**是**刷完一本讲义+一本习题（综合篇）**，或**二刷讲义+一本习题的重点章节**，未达到该要求或未结束强化的同学可以继续做题和听课。对于**基础比较好**（大学期间数学各科均90分以上）/**学习能力较强**（本科为985院校）的同学，最晚的冲刺开始时间是**11月初**；对于**一般同学**，最晚的冲刺开始时间是**10月初**；请各位同学参照这个时间点合理安排自己的复习进度。对于**冲刺开始时间≥上述时间点**的同学，**重点**是真题及针对真题中错题的查缺补漏，**辅助**是模拟卷（李四、李六等）；对于**9月份开始冲刺**的同学，真题和模拟卷都要重视，每份试卷都按照正式考试的流程认真完成。预计**9月17号（周天）**和大家见面，敬请期待。1. **数学二：****9月17号（周天）****见面**** 前提是刷完了一本习题➕一本讲义（最低要求），最好是刷完两本习题➕一本讲义****，进度相对较慢或者觉得自己掌握还不太行的同学可以继续刷习题，然后十月中旬再开启冲刺即可，切忌和他人攀比，不要看着别人进入冲刺了自己也着急忙慌地要开始。十月开始冲刺的同学也按着9月的广播进行下来就好，忽视掉里面的支线任务就OK，因为九月中旬开始冲刺进度比较快，之后会做市面上大部分模拟卷，十月中旬开始冲刺就可以选择一些重点模拟卷刷，主线和支线的划分可以参考《数二百日冲刺执行手册》**1. **英语一：暂未更新（由于近几日学长学姐刚刚开学，事务繁多，希望大家能够理解）**1. **英语二：**由于之前发过一篇帖子提问大家英二小三门有没有开始，评论区回答都是没有开始，那么英二冲刺贴我把小三门学习安排进去了，所以咱们得早点开始！！！预计**9月10号****（周天）跟大家见面**。使用英二百日冲刺广播贴的**前提是****至少一刷完英一大部分年份+英二16年之前的阅读理解（如果你的任务超过这个限定也没关系，因为我们每日规划的第一项任务永远不变，一直都是****每天一篇阅读****，只要你按照自己的进度每天一篇阅读即可），并且作文刷完了基础写作和强化写作(也就是小作文和大作文课)。****另外，想要跟大家说明的是我们规划实际开始是在9月15日（冲刺第一周周五），但广播贴发布时间必须定在前一周周天（即：9月10号），这也给了大家缓冲时间，希望大家利用9月10号~9月14号这几天的时间把进度拉到使用广播贴的最低要求！！！**1. **政治：**使用政治百日冲刺手册的建议**前提**是做过这一章的肖1000，政治这门学科有一点特殊，因为各个章节基本是独立的，所以**并不要求所有章节都做完肖1000再开始冲刺**，比如做完了马原，但是史纲才刚开始，那也完全可以跟着冲刺广播来巩固马原这一部分的知识，所以落实到每位同学自身的进度可能并不完全相同，大家进度不一样也不需要焦虑，有些同学可能先学马原、有些同学可能先学的史纲，我们只要抓住冲刺的重点就好：**重要考点和大题的背诵**。考虑到大家听视频课和做1000题需要一些时间，所以政治百日冲刺执行手册预计在**10.8号（周日）**和大家见面，如果觉得时间不太合理，请评论或私聊，我们会及时做出改进。最后，希望大家在这段时间把进度尽可能拉到进入冲刺的最低要求，政治这门课在后期才开始，大家不要担心，任何一门学科的学习方法都是大道至简，抓住复习重点，肯定肯定没问题的。1. **408（由于自命题情况复杂，这里暂时对统考进行广播）**：使用408冲刺手册的建议**前提是完成了408基础阶段和强化阶段的学习**，即至少完成408王道书上的所有选择题和除近三年之外的大题中的所有真题。完成这些之后，大家会对所有408的相关考点有比较全面的了解，对其中的一些重难点有比较深刻的理解。因为大家408复习的进度差别较大，有的同学可能强化阶段已经接近结束，有的同学可能强化阶段还没有开始。所以为了尽可能的照顾绝大多数学生，我们的百日冲刺手册预计**10月22号（周日）**与大家见面。至于进度比较快的同学，可以对王道选择题进行二刷，并同步制作思维导图，让自己对408的所有考点有更加深刻的理解。同时大家也可以关注星球中的408每日一题，里面有对408中考点的详细介绍，可以帮助大家更好的理解很多知识点。
希望大家看到上文后，提前把进度拉到使用广播贴的前提条件。另外，对于进度快于最低前提条件的同学，以星球广播贴为尺，自己向前跑就行。再次**希望同学们每周在冲刺广播贴下写下****本周的总结和反思****，学长学姐会根据大家的反思给出相应建议，帮助大家改进！！！**
