
# 完型方法论---来自易熙人老师课堂总结

本篇完型方法论来源于**elegant****的****易熙人老师**，以下内容只是对易熙人老师课程方法论的总结。
1. 阅读全文首句，**首句不设空，且与主题紧密相关**。1. 从语法角度确定空格成分（这一步是很简单的，因为**选项一般词性相同**，大家注意甄别多词性长相相同的单词，**那些你平时认为是名词的词，可能有动词的含义，且意思并不相近**，**而往往这种就是正确答案！！！**注意：当你已经排除其他三项不可能是正确答案时，而这一项词性在你认知中与其他选项词性并不相同，那么此时**往往最不可能的就是真相**，完型里有好几个这样的例子）1. 从语境中推测空格含义，找到最佳答案。
<br>

**一些小技巧：**
1. 若发现在一句话中设有两个空，顺着做逻辑不通顺的话可以倒着做1. 填动词的时候可以根据及物、不及物进行排除1. **简单词原则**，不知道选什么的时候，就选在认知里看起来最简单的单词1. 另外，**完型中选项ABCD出现次数不会差距太大（一般情况下，****每五个选项中ABCD四个选项会各出现一次****）**，就是说不会出现某个选项只出现1到2个的情况，如果真出现了这种情况，要好好斟酌一下不确定的空1. 如果两个句子间表示的对某事/物的态度相反，而句间又没有逻辑转折词连接时，逻辑词后移1. **对应暗示**（这个比较简单）：如果段落内容为总-分结构，即**归纳演绎，分句中形式相同，找句子结构相同点，对应找暗示**即可（这种空要是错，可别怪别人嘲笑你）1. 完型中至少会有一个表示转折词的空（比如：however、yet等等）
<br>

后续宝典中会总结一下**相同逻辑、相反逻辑的关系词**，以及易熙人老师总结的**皇后词**（大概率会选）、**宫女词**（大概率不会选，**老师叫洗脚婢的词是一定不会选的哈哈哈**）等等，大家可以关注下咱们的**英语必胜宝典文件夹**。
