
# 英二真题三种刷题顺序

**做题顺序1：****小作文——大作文——阅读——小三门****【小三门顺序建议：翻译（英二翻译简单，缓解一下前面的紧张）——新题型（给分题）——完型（就算最后没时间，蒙蒙也能拿2-5分，但我还是建议把握好时间，因为完型近几年也是给分题）】**（适合考前背作文模板的同学）

<br>

**做题顺序2：****阅读理解——小作文——大作文——小三门**（适合阅读理解很强+自己老老实实模拟过很多遍作文+怕碰到主题词不会情况的同学）

<br>

**做题顺序3：****阅读理解前两篇——小作文——阅读理解后两篇——大作文——小三门**（适合考场上脑子容易混沌的同学，阅读理解和作文交替做换换脑子）
