
# 飞小杰的自我介绍

        应寅贝勒邀请，加入了咱们计狗上岸的知识星球，很快乐能和大家一块分享自己的一些心得体会，本人本科双非，在2018年以成绩推免西电（2017年曾短暂的复习过考研），在计算机专业攻读硕士研究生，简要概括一下自己的经历，希望能对大家有所帮助，祝大家一站成硕。

        我从以下几个方面总结我的本科生活，学习，生活，科研，希望能给大家带来帮助

        第一，学习方面。在校期间我学习的科目有数据结构，操作系统和嵌入式的相关知识，包括理论性较强的计算机组成原理，实用性较强的嵌入式操作系统和嵌入式设计与开发等课程。这时候告诉大家一些小技巧，在平时期末考试前的紧张备考中，一定要多找老师去答疑，有一次我找老师去答疑，老师说就你一个人来问，就直接把考试题和我说了，所以和老师处理好关系也很重要，还有就是如果你现在纠结自己到底能不能保研，那你如果处于大二，可以大三多使劲，原来可能考试突击一下，凑乎个及格，但是现在可能平时每一节课都认真听，如果你处于大三，提前计算自己的绩点，看能不能达到要求；另外，建议大家一定要学好英语，因为你的四六级成绩，有些学校。会在拿你保研的过程中会卡你四六级成绩，如果你四六级成绩没有达到要求，那么你无论成绩多高也没有用，还就是在四六级考试过程中，你备考的过程中往往都是在学期末，所以说你可能与那个平时的考试也会有一些交集啥的，然后你平时可以多听一些那种新东方呀，还有就是那些考虫呀那些方面的一些视频课，积极备考，因为大学就八次考试嘛，所以说你在大学期间尽早过了就把这个事情结束了，如果没有过的话一直会留在后面，不太好作为一名对英语学习有极大兴趣的学生，我在大二就把四六级都过了，备考考研时候一直刷朱伟的刷词班恋练有词，如果大家想听老版本的恋练有词可以找我要，培养语感和平时积累也可以积累。

       第二，生活方面。我担任了四年班级学习委员，在各门课程的小组演示中，我也积极参与协作，经常担任主讲人，在各门学科的学习中，协助老师顺利完成教学任务，同时负责电脑社团的管理与运营工作，多次承担团日活动主持人，经常参加校级辩论赛，培养了较强的思辨能力。此外，四年来的核心班委工作更是提高了我的沟通协商能力。骑行，露宿开阔了我的视野；暑假支教与孩子们的接融，让我抛开浮躁和功利；做志愿者服务敬老院的老人，给了我给予大于得到的快乐。

       第三，科研经历方面。工科最注重学生的三个能力，编程能力，英语能力和数学能力，你如果平时想做一些项目的话，你可以去找老师要一些项目，你们可以给老师的研究生打下手呀，这些都可以。因为当时我们是有一些比赛，然后就比如说这些本科生参加的一些比赛，然后他们都是学长学姐们做过的，然后再接手过来，大二，大三的时候可以自己自己项目负责人的身份去申请一些项目，然后在那个当时做了两年的时间，成功结题，然后也可以去参加一些数学建模的一些比赛，再比如美赛呀这些赛事，然后可以提升自己的能力，然后还有就是可以提升自己的写文章的这种能力，这个也是在研究生期间必须要掌握的，我当时是参加了两个比赛和一个大创，有些学校大创会给额外竞赛保研的资格，还有就是参加数学建模也可以，主要还是提前去修炼一些技能吧，我参加过从本科生的物联网大赛《[基于ARM平台的智慧教室系统设计](https://wx.zsxq.com/mweb/views/weread/search.html?keyword=%E5%9F%BA%E4%BA%8EARM%E5%B9%B3%E5%8F%B0%E7%9A%84%E6%99%BA%E6%85%A7%E6%95%99%E5%AE%A4%E7%B3%BB%E7%BB%9F%E8%AE%BE%E8%AE%A1)》的尝试，后来以项目负责人的身份申请校级大学生创新实验项目《[基于Arduino平台的智能鱼缸系统设计](https://wx.zsxq.com/mweb/views/weread/search.html?keyword=%E5%9F%BA%E4%BA%8EArduino%E5%B9%B3%E5%8F%B0%E7%9A%84%E6%99%BA%E8%83%BD%E9%B1%BC%E7%BC%B8%E7%B3%BB%E7%BB%9F%E8%AE%BE%E8%AE%A1)》，到中国机器人大赛助老服务机器人助老生活服务项目赛激烈角逐，作品为《[基于Arduino平台的智能寻迹小车](https://wx.zsxq.com/mweb/views/weread/search.html?keyword=%E5%9F%BA%E4%BA%8EArduino%E5%B9%B3%E5%8F%B0%E7%9A%84%E6%99%BA%E8%83%BD%E5%AF%BB%E8%BF%B9%E5%B0%8F%E8%BD%A6)》。我在2016年9月参加了全国大学生数学建模比赛，在2017年4月参加了数学中国网络建模挑战赛，增加了项目经历可以往简历上写，对面试有加分项。

       最后，请允许我用《[功夫熊猫](https://wx.zsxq.com/mweb/views/weread/search.html?keyword=%E5%8A%9F%E5%A4%AB%E7%86%8A%E7%8C%AB)》中的一句台词来结束我的自我介绍：“你人生故事的开头也许充满坎坷，但这并不影响你成为什么样的人，关键看后来的人生路，你选择怎么走下去。”祝愿各位“计狗”能够成功上岸！
