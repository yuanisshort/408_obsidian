
# 1.单词（3-12月份）
单词贯穿整个英语备考过程。3-12月份每天都要背单词，一定是每天。根据自己的能力设定每天的单词量，确保在考试前要背完三遍单词。我用的是手机背单词APP+恋练有词，可以结合在一起背。

并不是要求我们要做到把这五千多个大纲单词全部正确的默写下来，能够让你有机会写单词的题型就只有作文，而你写作文会用到这么多单词吗？并不会。

我们需要做到的是熟悉所有的大纲单词，看到这些单词我们要知道它常见的意思是什么，不会写不要紧，只要你能做到能够认识阅读理解中的95%以上的单词就足够了。写作文需要用到的一些单词基本上就是高频单词了，高频单词你肯定就要会默写了，但这个任务量不会很大。

背单词的过程中忘了背、背了忘，这个是很正常的，我们的大脑又不是储存卡，所以才需要我们每天都背单词，及时地回顾之前背过的单词。

还有，完全没有必要去看单词视频课！！！！一个单词讲十几二十分钟，我觉得那就是浪费时间，我可以用这个时间背好几个单词了。我只是用了恋练有词这本书，并没有去看恋练有词的视频课，因为没必要。


**1. 单词—是做所有题型的基础！一定要从头背到尾！即使最后时间再紧张也要坚持背！**


首先大家要掌握大纲5500单词，然后再把历年真题阅读中的单词也要掌握。单词要多刷、快刷、反复刷，只有重复才能记住单词。

# 2.阅读理解（3-12月份）

阅读是考研英语的重点，阅读A节4篇共20分，B节10分，一共50分。如果阅读崩了那英语也就崩了，不过大家也不用太害怕，只要认真背单词、学语法，再把老师的阅读技巧课好好听几遍，再把历年真题做一做就能提高正确率。

**3.** **阅读**（40’）—在考研英语中占比最大，所以最重要！得阅读者得天下！这句话是不是从初中就开始听，对于研究生来讲英语阅读好看文献就快，不然总靠软件翻译，你的效率自然就比别人低。所以对于英二来讲，阅读最少32分，掌握技巧认真准备不难的。


阅读理解是最先复习的题型。我考的是英语二，但是英语一和英语二的真题都得做。先做英语一，再做英语二。

## 3-8月份我做的是英语一的阅读理解。

3-6月份我每天只做一篇甚至两天才能做一篇，就是要做慢做精。一开始接触真题时，一篇阅读要花一两个小时而且还会错好多，不要灰心，一步一步来（刚开始的时候千万不要追求做题的速度，这是你到后期才考虑的问题）。

田的阅读精读，你千万不要做完一套题，对个答案翻翻解析就直接去看视频，要先不查单词和翻译的情况下，对自己不会的句子反复读，看能否在不限时的情况下拆解长难句或者读懂作者想表达什么。实在不会的句子要标注，并且写哪里没读懂，自己的理解是什么，然后再去看他的精读课。因为事实上英语看句子和数学做题一样，只要有一点提示，比如说从句动词到底是跟的哪个主语，长难句的理解将变得特别简单。如果你是对完答案就去看他的视频，很容易先入为主用正确的方法，让你忽视对“独立分析”能力的培养和对自己错误的剖析。

我做阅读理解的步骤是：

1）浏览一遍题目和选项，把选项当中的不同和主谓宾圈出来。

2）阅读文章，一句一句地在心中翻译，生单词就划线，长难句就画出句子的结构（比如定语从句、状语从句啥的）。如果通读完了一遍文章完全不清楚这篇文章在讲什么，你需要从头开始再阅读一遍。

3）读完文章，开始做题。把每个题目在文章中出现的位置画出来，对照选项去做题。

4）做完了对答案。最好是买那种答案解析和原文翻译很详细的资料（推荐张剑黄皮书）。认认真真地看解析和原文翻译，看和自己想的地方到底哪些不一样。

5）确认自己看完答案后每个题都弄懂了，然后把这一篇所有的生单词和不认识的词组摘抄到笔记本上，注明音标和中文含义。


1. 首先阅读题目已预测文章内容，先读题目，再读文章，同时根据题目设想文章可能涉及的问题。

2. 通读文章，逐个题目进行解答，一篇文章大概需要15-20分钟。

3. 做完真题后，暂时不要对答案，将文章中所有的英语生词整理到一个本子

上，拿出真题再做一遍，做完第二遍后对答案，看能否理解为什么选择

4. 逐句进行文章翻译，分析文章中出现的长难句，圈出句子主干。

## 7-8月份
有的时候每天一篇或者两篇阅读理解，在8月底之前把所有的英语一阅读做完一遍，我当时还有剩余时间，所以把有的年份的英语一阅读又刷一遍。

## 9-12月份
做英语二阅读理解，此时我的正确率和做题速度已经提高了不少。基本上是每天做1-2篇阅读理解。把最新两年的英语二真题留着不要做，等到考试的前一两个星期再做，并给自己计时。

我大概是11月就把英语二阅读真题做完了，此时我就开始二刷三刷了。如果有的题我连错了两遍，我就会去思考这种类型的题我到底是哪里没想通，思维错在哪里。即使到了考前前几天，也要每天做一篇阅读保持手感。


有的同学会问那应不应该看阅读视频讲解呢？我说说我个人的看法：我只看过两次阅读视频，是在七八月份有段时间我的阅读正确率总是提高不了，我就去看了。此前我的基础是会划分一个长难句的结构，哪里是状语从句、定语从句等等我基本都知道。

但是看了视频后我觉得老师讲的和我想的差不多，所以我就没有继续看了，而是从其他的方面寻找自己正确率提高不了的原因。如果你目前的状况是不知道怎么去分析句子的结构，那我觉得看长难句讲解视频是很有必要的，主要推荐唐迟老师和高晓艳老师的视频。

# 3.完形填空（9-12月份）

完形填空不用开始得太早，9月份开始就可以。每天或每两天做一篇完形填空，大概十一月份之前就可以把英语一和英语二的完形填空全部做完。我做完形填空的方法是：不要到了一个题就马上去选答案。

首先是通读全文，读到了每个题号处时看一眼该题的所有选项，代入原文看看，心里大概有个想法和印象就行。读完了全文，然后再从头开始读，开始做题了，犹豫的题放到最后再做，先把确定的写上。

英语二的完形填空要比英语一简单得多，没有什么技巧，我觉得主要还是看语感。所有的完形填空做完一遍可以开始二刷了，主要是保持手感。

# 4.新题型（11-12月份）

考英语二的同学们，只用做英语二的新题型，不用做英语一的不用做英语一的不用做英语一的！！！英语二的新题型比较简单，通读一遍了解个大致意思基本就没什么问题了。阅读理解都做过来了还怕什么新题型呀！

 **翻译+B型阅读+完型**（15’+10’+10’）—英二的这三个部分不难，完型可以跟着阅读一起做，好多人都说这部分难放弃，不可绝对不可，硬着头皮练，7/8分没问题。翻译需要听课，可以跟着大阅读一起练，练得好可以拿到12分。新题型是最简单的，前期基础打好了后期直接上手，仔细做可以拿满分。



这三个也叫小三门，是相对简单一点的题，可以直接做，时间充足的话也可以听一下老师的技巧课。



推荐做题顺序：作文-->阅读（AB）-->翻译-->完形



大部分人在做完形时没剩多少时间了就把完形蒙一蒙，所以大家也不要花太多时间放在完形上，先把阅读、作文搞好，这样完形随便写才不会影响分数，当然，如果想考高分，也抽时间把完形练一练。


# 5.翻译（11-12月份）

11月份开始翻译就行。英语一是翻译指定的几句话，英语二是翻译给定的一段话，而且难度远远不如英语一。我建议是看看英语一的翻译就行，也不需要你练手，把英语二的翻译做好就行。不要太口语化，稍微优美一点就ok。

# 6.作文（9-12月份）

作文资料我就只买了王江涛的《考研高分写作》。9-12月份每天背一篇大作文模板，也不是说要完整地背下来，主要是要掌握整篇的写法以及一些通用的句子。脑袋里记得好句子越多，考试的时候才能得心应手。

10-12月份每天背一篇小作文。9-12月份每天自己写一篇作文（后期可以写两篇），严格控制时间。我写了估计有100多篇了，最后考试的时候写得很快，感觉一些好句子在脑袋里迫不及待地要跳出来了！

一步步走来，我觉得自己学得很踏实，自己的努力对得起最后的结果。在这里把我的英语备考经验完整地分享出来，希望你们可以学得更轻松，拿到更高的分数！

简而言之，就是不要偷懒，把每一个题型都练到位了，最后的考试结果一定不会太差！如果有的同学做得比较快，那就继续把真题二刷三刷就可以，不用做模拟题，只用做真题！冲啊集美兄弟们~

**4.** **作文**（10’+15’）—分为小作文（书信类）和大作文（议论文）两种，重要性仅次于阅读，一般来讲认真准备17/18分不难，其实大家也都这个水平，写作要是想拿高分，那是需要很扎实的基础，短期内这部分的能力增长不会太快，但是也不能偷懒，背默写一样也少不了，不能就靠最后一个月背背模板和押题就草草了事，老师看得出来是怎么回事。



作文在10月开始就可以，大家可以听一下石雷鹏老师的作文课，然后自己总结一些模版，照着历年真题的作文题目自己尝试写一下。



如果自己写的不太行或者根本写不了，那就买模版背作文模版。有人肯定就会说，这些模版不靠谱，重复度高，用了分数很低。模版确实生硬一些，但是总比在考场上一句写不出来强很多，网上也有299元12个或者10个大小作文的模版，保证不重复，不过这个是不是夸大宣传也就不清楚了，买模版是很能解决问题的方法，所以大家也不必焦虑。
