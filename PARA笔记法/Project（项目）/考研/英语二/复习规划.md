# **1.** **3~5月**：

大部分人正式开始考研复习应该都是在3月份，在基础阶段英语需要解决的是单词和语法长难句+小三门。对于单词来讲，我个人建议**不需要看视频**，比较浪费时间，甚至也不需要买单词书，因为背单词的APP里都有，这里我推荐**扇贝和[墨墨](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=墨墨&zhida_source=entity)**。



**扇贝**可以免费选单词书，如果你已经买了实体单词书，在里面也可以选。基本上就是恋练有词/[红宝书](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=红宝书&zhida_source=entity)/十天搞定。如果没有书，系统会根据你的备考阶段生成单词书。刚开始一定要把核心词（2000左右）背熟，接着5000多个大纲词过1~2遍，到后面就可以重点抓不熟的词和核心词背了。单词下面也会有真题例句，词根词缀等。可以自己设置学习任务和天数，40天左右过完一遍。对于之前基本没背过单词或者一点也不了解**词根词缀**的同学来讲，是有必要先把这个技巧学会的，可以听听课，去B站和微博上搜，什么资源都有（可以听听[周思成](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=周思成&zhida_source=entity)，据己情况及时调整），记得倍速！



如果你觉得掌握的差不多了，其实也不用花那么多时间了，还是以专业课为主。单词记忆主要还是**利用碎片时间**，比如早餐时，看专业课书累了时，睡觉前。卡子姐的单词带背音频可以听一听，感觉记忆的效果也是很好滴。


清明节前
长难句这块我建议听听有道**田静老师的语法课**，她会特别系统性的让你掌握语法的逻辑。只要能把那棵[语法树](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=语法树&zhida_source=entity)印在脑子里，应对考试是足够的。对于[虚拟语气](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=虚拟语气&zhida_source=entity)、倒转插入句这种比较难的小枝节不用深究，只要把**各类从句、[非谓语动词](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=非谓语动词&zhida_source=entity)、时态语态**等大树干搞明白了就行。跟着课都听明白了，可以自己分析一下长难句，多练就会拆了。句子中遇到的生词可以用墨墨记忆。其实可以**直接上手练真题**，不用做题，就拆分句子看懂即可。

田静的语法长难句课是非常优质的，看了你会有明显的感受，她和刘晓燕是两个极端，**一个不善于考玩笑，全程干货，有时候甚至都能让你学累了，因为知识点太饱满了，得留着课下吸收。另一个就是笑话太多，以至于你得课下放松放松，按摩按摩，苹果肌太酸了。语法是贯穿整个英语乃至是考研英语的核心，从小学初中我们就开始接触语法，但你现在还记得多少，系不系统？长难句里面的各种成分能不能清晰判别并划分好呢？如果不能，那么可以适当看看语法课，效果简直不要太好！

前期（清明节后）到7月中旬可以**1~2天一篇阅读**(蓝皮书)+B站ELI田汉博（**在真题中学语法和背单词**）
遇到不认识的单词或者语法多读几遍，查意思后第二天起来继续看，知道看会为止
考试是用来检验的，而不是提升自己的



# **2.** **6~9月**：

前期打好一定的基础，中期需要解决的是**阅读+小三门**，暑期是你与别人拉开差距的黄金时期，尽管很难，但只要能啃下这块硬骨头，等到10月左右，一定会感觉英语得心应手！我建议先做英一的阅读和完型（其他的都不用做），04~21年的真题都要练，一天两篇，40天左右过完一遍。

**6~8月学知识**
### **(小三门)**
说说小三门，主要包括**完型、翻译和新题型**，这一块着重强调可以看看刘晓艳的，而且**她这部分讲的是最好的**。因为小三门是很多考研人比较容易忽视的部分，**完形填空一般都会放到最后顺序去做**，时间不够的话干脆草草了事，翻译其实我们并不陌生，四六级就有，但两者相不同的是，考研翻译是英译汉，四六级是汉译英。英译汉容易出现下述问题，**中心词不认识**，这就很尴尬，基本代表这一块彻底拉跨，翻译的都不对，还有就是对于语序语法长难句的问题，能不能正确书写出既规范又优美的汉译。新题型这一块其实并不难，多做做真题，感受感受就没大问题。

小三门这一块并不是“**鸡肋**”，如果说只为了英语过线，**拿个四五十分**，那注意力完全可以放在**【阅读】+【作文】**部分，反之，如果想拿高分，那小三门是不得不必修的重要模块，刘晓燕的小三门课程，好好学一学，不要浪费太多时间，课上勤记笔记，**一些她说的段子水文，适当跳过**，这些东西一是影响注意力，二是和提分一点关系也没有，学完立刻投入到做题部分。（其实我们考研人看网课的目的很明确，就是为了辅助提分，要是看段子我为什么不去刷小视频呢？）


**新题型刘晓艳**阅读是最简单的，一般就是七选五/给段落选标题，看仔细了基本都不会丢分，毕竟前期经过阅读的淬炼，你的英语水平肯定是质变了。

**完型易熙人**我是跟着阅读一起写的，我没觉得特别难，一开始肯定错的多，多练就熟了，要是把完型放在最后去准备，首先是时间不多了这项见效慢，再加上各科复习也都紧张了，很容易就把它忽略或是放弃了，因为确实分值不高，但那也是分啊！**多一分你可能就挤掉好几个人**！这个阶段阅读不能停啊，至少一篇，保持做题手感！

**八月中旬之后做真题！**

早上做题，下午看解析加总结。可以准备一个笔记本，要把每道题的类型写下来，错题要写明错的原因（单词不熟/找错句子/关键句没看懂）不熟的单词一定都要记住，墨墨就好在可以编辑自己的单词卡片（词组真题熟词僻义等），建立自己的单词库，也是按照个人的记忆情况进行复习。

在笔记的最后要把**整篇文章的逻辑**写下来（每段的大意）一般每篇阅读的最后一题都会问主旨题/选题目，所以需要理解和概括全文。多锻炼聚合思维这类题就会写了。还有一种错的比较多的题型是细节题，这个一般就是考察长难句、同义替换，多总结，找规律。

可以拿那种[手译本](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=手译本&zhida_source=entity)精读，记录idea。重难点的长难句，可以自己手动翻译，这样翻译能力也会不知不觉的提高了。另外真题中出现的好句好词可以抄下来，以后放在作文里用。



**英一刷完一遍开始英二，恭喜你进入简单模式！如果还是正确率不高，不要着急，还没到质变，英二一遍完大约在8月中上旬。10~19年的题跟上述一样精读，记笔记总结。



再来一轮英一英二，基本上就差不多了。可以用铅笔写后面再擦，也可以直接买一套题，无解析的那种10块左右吧。这轮会快一些，可以**直接用你的笔记**进行订正，如果有新错的题再去翻解析。据我个人经验讲，错过一遍的题很有可能还错，所以可能是你解题的思维和标答的思维不一样，需要自己思考，顺着那个思路想就选对了。如果看解析看不明白，可以听一听[唐迟](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=唐迟&zhida_source=entity)老师的阅读课，唐迟的课也不用都看，费时间，主要抓住他【**逻汉班**】就可以。一篇讲大约25分钟吧，倍速挺快的。

真题不在乎数量，而是要做透

标准：
- 几乎没有生单词
- 每句话都会分析和翻译
- 明确知道每道题为何错

# **3. 9~11月**：
**翻译**需要听听[唐静](https://zhida.zhihu.com/search?content_id=171162159&content_type=Article&match_order=1&q=唐静&zhida_source=entity)老师的课，我就是到这段时间主攻专业课也没时间听，以为自己能翻过来就行，但其中的采分点和语序调整不太清楚，所以可能答得不好。英二翻译一篇很简单的文章，比英一轻松多了。



这时候作文就要开始准备了，首先应该知道写作的思路，可以背一背陈曲（奶哥）。

都背完一轮以后，可以自己上手写英二的历年真题，先自己分析和思考，实在是没有思路可以看看那本书，学习总结一下。几乎近十年的题所有类型都会涵盖的。然后自己写完的文章，可以用网站-**微软爱写作**去订正，单词和语法的错误都会被查出来，还会给出一些高级替换的词。每一篇写完的都会被保存下来，方便以后查阅。个人感觉大作文比较好写，就是分为文化、教育、经济和环保这几大类吧，**总结一个自己的模板**，直接套就行。小作文有些灵活，需要根据题目所给信息写，不过不长也能套，万变不离其宗。

陈曲的核心思维就是**把作文想象成填空一样简单**，这点也是我当时非常看重的，因为后期留给作文的准备时间不多，再去死记硬背，上考场绝对自乱阵脚，给大家透个信，考试的时候没有人构思作文，最多就是阅读一下题目，直接开始写，甚至没有那种说先写到纸上，慢慢理顺，再腾到纸上的做法，**大多数人在上考场前其实就把作文准备好了**，什么格式？什么结构？怎么套用？作文部分就像[弹药库](https://zhida.zhihu.com/search?content_id=472659717&content_type=Answer&match_order=1&q=%E5%BC%B9%E8%8D%AF%E5%BA%93&zhida_source=entity)一样，特别符合应试理念，一点也不拖泥带水。

所以看九宫格的时候，一定要熟悉**几大模板+几大框架**，将它们掌握好之后，在自己的“**小天地**”里巧妙运用，可以使用你平时积累的美句美段，也可以直接按照陈曲给你准备好的话术去套用，九宫格非常灵活，也非常好用，你看了就知道了。看完九宫格也不代表你会了，最好是按照年份做历年真题，**把每个作文通通练一遍，比如英一的漫画作文（文化火锅等等），或者英二的图表作文**，属于什么类型？科技文？教育文？个人品质文？等等，练完一遍下来，你的作文绝对没问题，我就是从这条路走过来的，所以不希望大家走太多弯路（考研人时间很宝贵的）

还有就是快要考试的时候，会有各种**直播课/冲刺班/救命班**，我当时其实是没怎么看的，因为十一十二月份，注意力都放在政治和数学上了，英语这个东西就是**日积月累**的过程，每天给自己定目标，单词背多少，阅读写几篇，只要坚持执行，根本不需要这些课，后期英语这一块主要就是保持手感，你会发现几天不做题，之后再上手会有生疏感，*如果考试时间正处于这种陌生期，那么是非常不利的，而且临考阶段，英语部分的重点就在作文上，阅读等题型平时基本练的差不多，该是什么样都定型了，只有作文是可以大幅度提升的要点。

# **4.** **冲刺12月**：

这时候如果复习的好你可能感觉我英语没问题了，but！**还是保持每天基本的学习量！阅读写作都要坚持，月中各大老师的作文押题就出来了，自己整理一下，把重复率高的押题多写写，在规定时间内手写不能翻资料查词。然后剩了两年的真题拿出来模拟，三个小时从头到尾就像正式考试一样，至少要模拟两遍，拿那种答题卡写**。该阶段最重要的就是**调整好心态**！无论是哪一科，你都要相信自己已经完全有能力去应试了！模拟后对答案不要太在意得分，重点是错在哪里？怎么错的？模拟的最大意义是要找到考试的感觉！最重要的是要在该阶段**平衡好时间**，因为政治（背大题）和专业课在12月的复习节奏都非常紧张，很容易挤掉复习英语的时间，万万不可大意。稳住！我们能赢！

一定不要**查漏补缺**（女娲补天）,而是要去复习总结，把以前所有记得笔记，背过的所有单词以及真题、老师讲过的方法拿出来过一遍，做两套模拟卷,不要看新题，而是看过去的阅读理解

 **早完成早结束，晚完成晚结束，每天完成自己的计划！！！**
