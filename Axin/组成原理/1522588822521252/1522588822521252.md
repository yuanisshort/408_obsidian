先算时间，再算MIPS数。

序列执行时间 = 总的时钟周期数 x 时钟周期
序列1执行时间 = (5x1+1x2+1x3) x 10^9 x 2 x 10^-9 = 20 秒
序列2执行时间 = (10x1+1x2+1x3) x 10^9 x 2 x 10^-9 = 30 秒

MIPS = 执行的指令数 / 执行指令所用的时间（秒） / 1,000,000
序列1 = (5+1+1) x 10^9 / (2 x 10) / (1 x 10^6) = 350
序列1 = (10+1+1) x 10^9 / (3 x 10) / (1 x 10^6) = 400
