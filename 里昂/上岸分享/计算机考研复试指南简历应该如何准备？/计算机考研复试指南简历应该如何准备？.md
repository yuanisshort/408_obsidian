            <div class="content ql-editor"><p><span style="color: rgb(0, 0, 0); font-size: 15px;">简历也是复试必须准备的内容。无论你的报考院校允不允许带简历进复试，请各位都准备一份。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">复试用不上，联系导师的时候也是需要的吧！</span></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">所以请各位重视简历。</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;"><br></span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">简历上应有一些基本信息和一些能让老师对你“感兴趣”的内容。</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">一般简历都包括以下内容：</strong></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">1.个人信息</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">姓名、年龄、性别、照片、联系方式（手机号和邮箱都需要放上去）</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">2.教育背景</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">这个的话就是写自己的本科院校和所学专业，没有必要从高中开始写哈/(ㄒo ㄒ)/~~ </span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">另外，有部分宝子觉得自己的本科不是很优秀的话</span><span style="color: rgb(0, 82, 255); font-size: 15px;">，可以把这一部分的内容适当地放在简历比较不起眼的位置，</span><span style="color: rgb(0, 0, 0); font-size: 15px;">将自己有优势的部分占整个简历的大头。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">当然，请务必不要造假！觉得自己本科学校不是特别好，就换一个学校瞎写，一定要避免！</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">无论是现在给导师自荐也好、还是未来就业找工作也好，都尽量不要做这种事情~</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">3.初试成绩</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">如果初试成绩比较优秀的宝子，可以在简历中上部分</span><strong style="color: rgb(0, 0, 0); font-size: 15px;">着重强调一下自己的初试成绩</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">（尤其是对数学、专业课成绩比较理想的宝子）</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">4.竞赛及科研成果</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">4、5 都是老师更愿意看的部分，大家有的一定要记得写上去，</span><strong style="color: rgb(236, 40, 52); font-size: 15px;">告诉老师“我这个人能干活”！</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">学长本科的时候跟着老师做了几年科研，有一点小小的成果和竞赛经历，这些成果还是有些用的！</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">所以，如果是大二的宝子准备考研的话，完全可以去准备一些竞赛、科研。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">5.项目及实践经历</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">项目也是大家可以着重去写一些的部分，可以参考 STAR 法则进行准备（star法则同样适用于自我介绍哈）</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">把你的项目/实习的名称、负责板块、遇到的困难、解决方式和成果都简要概括，这样会便于老师去了解你。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;"><br></span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">此外，尽可能简历在一页纸上，你想想，其他同学的简历都是一张纸，你还要拿一个订书机/别针搞个好几页，属实有点扎眼，没必要。（当然！大佬除外！）</span></p><p><br></p></div>
