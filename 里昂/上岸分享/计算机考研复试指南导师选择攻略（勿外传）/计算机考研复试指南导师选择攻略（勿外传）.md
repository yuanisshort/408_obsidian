            <div class="content ql-editor"><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第一步，给导师们分类</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">常见的分类方式可以按照老师的“江湖”地位、行政职位和躺平指数进行划分~</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">但今天，学长想提供一个更加实用的分类思路，根据老师</span><strong style="color: rgb(0, 0, 0); font-size: 15px;">当下的科研水平和管理风格</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">进行分类。</span></p><p><br></p><p><strong style="font-size: 15px;">一、关注老师当下的科研水平</strong></p><p><span style="font-size: 15px;">首先，强调一下，是【当下】的科研水平~</span></p><p><span style="font-size: 15px;">计算机发展得实在太快了，很多老师</span><span style="color: rgb(0, 0, 0); font-size: 15px;">可能都来不及学习新的东西，所以大家一定要关注老师当下的科研水平和对科研的熟悉程度！这个很重要！</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">那如何判断老师的现在的科研水平呢？</strong></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">这里给大家提供一个思路：去翻阅老师近年有没有发表过CCFA类的英文会议或期刊（会议难度高于期刊）</strong></p><p><br></p><p><span style="font-size: 15px;">如果没有，那老师工作的重心可能就不在科研上了，带你冲击顶会的概率也会相应比较小。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">（当然不排除有的老师比较年轻，30 出头，还没有搞出成果来。同样，也要提防这样的年轻老师，因为很多老师在这个时间段一直没成果的话就会放弃科研去搞钱搞关系了……）</span></p><p><br></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">此外，还要注意，如果这个老师只有分散的几篇 A 刊，而且都是一两个学生发的，那有可能是学生</span><span style="color: rgb(236, 40, 52); font-size: 15px;">带飞</span><span style="color: rgb(0, 0, 0); font-size: 15px;">老师！</span></p><p><span style="font-size: 15px; color: rgb(0, 0, 255);">如果你是追求学术，渴望在学术界建功立业，你懂的……</span></p><p><span style="font-size: 15px; color: rgb(0, 0, 255);"><br></span></p><p><strong style="font-size: 15px; color: rgb(38, 38, 38);">二、关注老师的管理风格</strong></p><p><span style="font-size: 15px;">关于管理风格，学长比较冒昧地分为了三类，分别是【认真负责的】【不管的】【瞎管的】</span></p><p><br></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">首先，是</span><span style="color: rgb(236, 40, 52); font-size: 15px;">【认真负责】</span><span style="color: rgb(0, 0, 0); font-size: 15px;">的导师，遇上这一类导师，恭喜你，</span><u style="color: rgb(0, 0, 0); font-size: 15px;">基本上再摆，跟着老师的要求也能混出点名堂来</u><span style="color: rgb(0, 0, 0); font-size: 15px;">。当然，大家也要稍微界定下这个度，如果有的老师【极度 push，到了压榨】那个层面上，还是要根据自己的抗压能力和心理状况看看要不要趁早跑路。</span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">其次，是</span><span style="color: rgb(236, 40, 52); font-size: 15px;">【不管】</span><span style="color: rgb(0, 0, 0); font-size: 15px;">的导师，</span><u style="color: rgb(0, 0, 0); font-size: 15px;">这一类老师通常是行政职务比较高、有自己公司或者已经看淡学术风云选择躺平的大佬</u><span style="color: rgb(0, 0, 0); font-size: 15px;">。跟着这一类导师，需要各位对自己严格一些，把三年读研的时间作为就业的踏板，研一就开始学习一些相关知识，研二研三就可以疯狂找实习了。所以如果你是以就业和实习为导向的话，建议直接选这一类导师~ </span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">最后，是</span><span style="color: rgb(236, 40, 52); font-size: 15px;">【瞎管的】</span><span style="color: rgb(0, 0, 0); font-size: 15px;">导师。这样说可能不太严谨，但确实存在这样的导师。遇上这样的老师大家就自求多福了。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;"><br></span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">那这个怎么判断呢？</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">建议大家去查看这个老师近年的一作和通讯论文，</span><strong style="color: rgb(236, 40, 52); font-size: 15px;">判断老师在研究方向上是不是聚焦的。</strong></p><p><br></p><p><span style="color: rgb(0, 0, 0);">如果出现 4、5 个学生都是不同方向的，有的在做机器学习、有的在做网络……这个时候大家就要警惕一些，你想想，一个老师再牛也很难有精力同时深挖很多个领域吧？</span></p><p><span style="color: rgb(0, 0, 0);">说明这个老师在接项目或者申报的时候，就比较随意，有啥接啥。</span></p><p><span style="color: rgb(0, 0, 0);">或者就是这位老师的研究生们已经意识到老师给的方向不靠谱后，各显神通了……</span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">三、了解导师当下的研究方向</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">另外再强调一下，了解导师当下的研究方向也很重要。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">一定要是</span><strong style="color: rgb(0, 0, 0); font-size: 15px;">当下！</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">你在学校官网老师个人主页中看到的研究方向不一定等于老师当下研究的方向！比如有的老师之前可能在做偏学术的研究，你进去了发现做的是全是和公司合作的项目开发。</span></p><p><span style="font-family: inherit; font-size: 15px;font-style;font-variant-ligatures;font-variant-caps; color: rgb(0, 0, 0);">这个的话就需要大家联系对应老师实验室的学姐学长了解一下具体情况！</span></p><p><span style="font-family: inherit; font-size: 15px;font-style;font-variant-ligatures;font-variant-caps; color: rgb(0, 0, 0);"><br></span></p><p><span style="font-family: inherit; font-size: 15px;font-style;font-variant-ligatures;font-variant-caps; color: rgb(0, 0, 0);"><br></span><strong style="font-size: 15px; color: rgb(0, 0, 0);">第二步，明确自己的需求和能力，缩小选择范围</strong></p><p><br></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">在能力的预估上希望大家更诚恳一些，</span><span style="color: rgb(0, 82, 255); font-size: 15px;">“自认为自己很强”可能会是你读研痛苦的开始。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">当然，这个还是要看你的具体情况，学长只是做一个提醒。而从读研需求上划分，可以把大家的读研诉求分为三类：</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第一类，就是来搞科研的，就是想读博的</strong></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第二类，就是来读个硕士的，想利用这几年搞实习、搞点人脉关系的</strong></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第三类，就是啥都没想好的，想着先进去了再说，让我做牛做马都行的……</strong></p><p><br></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">根据三种读研需求和划分老师的两个维度，学长给出以下建议：</span></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">如果你想搞科研、想深造，优先选：</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">能力强+认真负责型导师，其次选：能力强+不管的导师，最后选：能力一般+干预过多的导师也就是优先考虑大牛导师，比如说院士、杰青、优青、长江学者……</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">尽管这一类大牛可能比较忙，很多情况会安排给团队里的小导带，也就相当于加入了一个“公司”，由老员工来带新员工，而不会让老板亲自下场。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">但是大牛毕竟是大牛，名气大、头衔大、人脉广，对你未来的深造也好、就业也好，都有助力作用。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">（注：不绝对！不是所有的大牛都会帮忙推荐！带来积极效应！修行靠个人！）</span></p><p><br></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">如果你想搞钱、搞实习，优先选：</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">能力强+不管的导师，其次选：能力一般+不管的导师，最后选能力强+认真负责的导师也就是以找一个弹性上下班、不打卡、相对自由的实验室和相对开明的导师为首选</span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">如果你啥都没想好，只是觉得需要读这个研的话，学长只有一个想说的</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">：如果你能力不强，或者是跨考小白，尽量避免选择【瞎管】的老师，大概率你会很痛苦……</span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><span style="color: rgb(0, 82, 255); font-size: 15px;">在这一步的时候，大家就可以列出 1-3 个心仪的老师名单了，切忌，不要一棵树上吊死！</span></p><p><span style="color: rgb(0, 82, 255); font-size: 15px;"><br></span></p><p><strong style="color: rgb(38, 38, 38); font-size: 15px;">第三步，深度了解心仪导师</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">也就是进行最终的确认，确认包括：</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">1.联系该导师实验室的学姐学长，获得老师更具体的信息，了解实验室当下，或者说你进去了是要干啥的</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">2.联系导师，给老师做自荐。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">这里如果有的学校有明确规定，不允许提前联系导师的话就按照规定来哈。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">如果没有的话，各位可以给心仪导师发邮件表一下态，根据老师的回复看看他对你有没有兴趣，进行一个双选。 </span></p><p><br></p><p><span style="font-size: 15px; color: rgb(0, 0, 255);"><br></span></p><p><br></p><p><br></p><p><br></p></div>

font-size: 15px;font-style;font-variant-ligatures;font-variant-caps; color: rgb(0, 0, 0);">这个的话就需要大家联系对应老师实验室的学姐学长了解一下具体情况！</span></p><p><span style="font-family: inherit; font-size: 15px;font-style;font-variant-ligatures;font-variant-caps; color: rgb(0, 0, 0);"><br></span></p><p><span style="font-family: inherit; font-size: 15px;font-style;font-variant-ligatures;font-variant-caps; color: rgb(0, 0, 0);"><br></span><strong style="font-size: 15px; color: rgb(0, 0, 0);">第二步，明确自己的需求和能力，缩小选择范围</strong></p><p><br></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">在能力的预估上希望大家更诚恳一些，</span><span style="color: rgb(0, 82, 255); font-size: 15px;">“自认为自己很强”可能会是你读研痛苦的开始。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">当然，这个还是要看你的具体情况，学长只是做一个提醒。而从读研需求上划分，可以把大家的读研诉求分为三类：</span></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第一类，就是来搞科研的，就是想读博的</strong></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第二类，就是来读个硕士的，想利用这几年搞实习、搞点人脉关系的</strong></p><p><strong style="color: rgb(0, 0, 0); font-size: 15px;">第三类，就是啥都没想好的，想着先进去了再说，让我做牛做马都行的……</strong></p><p><br></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">根据三种读研需求和划分老师的两个维度，学长给出以下建议：</span></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">如果你想搞科研、想深造，优先选：</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">能力强+认真负责型导师，其次选：能力强+不管的导师，最后选：能力一般+干预过多的导师也就是优先考虑大牛导师，比如说院士、杰青、优青、长江学者……</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">尽管这一类大牛可能比较忙，很多情况会安排给团队里的小导带，也就相当于加入了一个“公司”，由老员工来带新员工，而不会让老板亲自下场。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">但是大牛毕竟是大牛，名气大、头衔大、人脉广，对你未来的深造也好、就业也好，都有助力作用。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">（注：不绝对！不是所有的大牛都会帮忙推荐！带来积极效应！修行靠个人！）</span></p><p><br></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">如果你想搞钱、搞实习，优先选：</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">能力强+不管的导师，其次选：能力一般+不管的导师，最后选能力强+认真负责的导师也就是以找一个弹性上下班、不打卡、相对自由的实验室和相对开明的导师为首选</span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><strong style="color: rgb(236, 40, 52); font-size: 15px;">如果你啥都没想好，只是觉得需要读这个研的话，学长只有一个想说的</strong><span style="color: rgb(0, 0, 0); font-size: 15px;">：如果你能力不强，或者是跨考小白，尽量避免选择【瞎管】的老师，大概率你会很痛苦……</span></p><p><span style="color: rgb(0, 0, 0);"><br></span></p><p><span style="color: rgb(0, 82, 255); font-size: 15px;">在这一步的时候，大家就可以列出 1-3 个心仪的老师名单了，切忌，不要一棵树上吊死！</span></p><p><span style="color: rgb(0, 82, 255); font-size: 15px;"><br></span></p><p><strong style="color: rgb(38, 38, 38); font-size: 15px;">第三步，深度了解心仪导师</strong></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">也就是进行最终的确认，确认包括：</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">1.联系该导师实验室的学姐学长，获得老师更具体的信息，了解实验室当下，或者说你进去了是要干啥的</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">2.联系导师，给老师做自荐。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">这里如果有的学校有明确规定，不允许提前联系导师的话就按照规定来哈。</span></p><p><span style="color: rgb(0, 0, 0); font-size: 15px;">如果没有的话，各位可以给心仪导师发邮件表一下态，根据老师的回复看看他对你有没有兴趣，进行一个双选。 </span></p><p><br></p><p><span style="font-size: 15px; color: rgb(0, 0, 255);"><br></span></p><p><br></p><p><br></p><p><br></p></div>
