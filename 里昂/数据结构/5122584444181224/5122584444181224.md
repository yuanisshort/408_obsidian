typedef struct lnode{
    int data;
    struct lnode *next;
}lnode,*linklist
bool listinsert(linklist &l,int i,int e)	//插入操作
bool listinsert(lnode*l,int i,int e)	//插入操作
这里的lnode和*linklist什么意思呢  为什么插入操作可以有两种表达
二轮复习的时候发现这里有些困惑
LNode指这个结构体类型，*linklist是指向这种结构体类型的指针。
在C语言中,typedef关键字用于为现有的数据类型创建一个新的名称。它是一种简单的类型定义机制,可以使代码更容易理解和维护。
