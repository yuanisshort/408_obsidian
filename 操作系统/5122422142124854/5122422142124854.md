第八题的答案看不懂hh
这个原答案是这样的：每秒产生120个时钟中断，每次中断的时间为1/120~ 8.3ms，其中中断处理耗时为500us， 那么其开销为 500us/8.3ms=6%。这个算法答案四舍五入后正好正确，但其实是瞎算的，这么算的话意思是1秒钟CPU全拿来处理120个中断了，其他事情都不要干了。

应该这么算：

已知:
- 处理一次中断平均需要 500 微秒 (0.5 毫秒)
- 定时器每秒发出 120 次时钟中断
我们需要计算操作系统分配给时钟中断处理程序的 CPU 时间比例。
首先,我们需要计算每秒钟处理中断的总时间:
- 每秒钟处理中断的总时间 = 120 次 × 0.5 毫秒 = 60 毫秒
然后,我们需要计算整个 CPU 时间:
- 每秒钟总 CPU 时间 = 1000 毫秒
最后,我们可以计算出操作系统分配给时钟中断处理程序的 CPU 时间比例:
- CPU 时间比例 = 60 毫秒 / 1000 毫秒 = 6%
所以操作系统将 6% 的 CPU 时间分配给时钟中断处理程序。![图片说明](D:\学习资料\考研\操作系统\5122422142124854\214421418225411.jpg)
